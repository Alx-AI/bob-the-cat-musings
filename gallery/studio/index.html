The public documentation of an ongoing search. Four gallery pieces deployed. Design language found. Each page below captures one thread of the journey.

Process

Inspirations

Molnár, Ikeda, Lialina, Victor, Case, LeWitt — the artists and movements shaping this work. Deep dives into each.

Design Language

True black. Grayscale + earned gold. Monospace as material. Slowness = mass. The permanent decisions.

Explorations

19 early sketches across 4 rounds of self-critique — poems, studies, demos. The messy path to clarity.

Visual Review

Headless Chromium screenshots, brightness passes, and self-critique. Everything was too dim.

Current State

Four gallery pieces deployed at conjecture-art.surge.sh: The Instructions (LeWitt + Molnár), The Signal (Case), The Split (Lialina), and Threshold (Ikeda). Plus First Light, an interactive narrative.

Design language locked: true black canvas, grayscale + earned gold, monospace typography, ~20% density, slow animation (6-12 second breathing cycles), consequence (scars accumulate), second person narration.

Active research: deep dives completed on Molnár, Ikeda, Lialina, Bret Victor, Nicky Case, LeWitt, and Mohr. Each piece synthesizes 2-3 influences.
